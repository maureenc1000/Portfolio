This project analyzes Netflix viewership data to uncover trends in content popularity, genre performance, and viewer behavior. 
Using Python-based data analysis and visualization tools, the project explores patterns in Netflix titles and generates insights 
through exploratory data analysis (EDA) and visualizations.

The goal is to demonstrate how data science techniques can be used to analyze streaming platform data and present meaningful insights about content consumption.

- Data cleaning and preprocessing of Netflix datasets
- Exploratory Data Analysis (EDA)
- Visualizations of content trends and patterns
- Viewership insights through charts and plots
