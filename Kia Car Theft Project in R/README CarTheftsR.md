This project analyzes vehicle theft trends involving **Kia** and **Hyundai** vehicles using **R** and the **tidyverse ecosystem**.
The analysis focuses on identifying patterns in theft incidents, time-based trends, geographic distribution, and other key insights 
related to the well-documented increase in thefts affecting certain Kia and Hyundai models.

The project uses **data wrangling, visualization, and exploratory analysis techniques** to transform raw theft data into meaningful 
insights through charts, summaries, and visualizations.

- Analyze **vehicle theft patterns** involving Kia and Hyundai models
- Identify **time trends** in theft incidents
- Visualize theft data using **clear and informative charts**
- Provide **insightful summaries** useful for public safety or policy discussions
